# Recorded result snapshot

These files are generated by:

```bash
python studies/fuzzy_rough_reference_study/run_study.py
```

They are committed as a reproducible research artifact rather than as universal
performance claims. `environment.json` identifies the exact execution context,
and `study_manifest.json` contains SHA-256 checksums for the generated files.

After merging changes that affect the public API, model computations, benchmark
harness, study configuration, or dependency versions, regenerate this directory
from a clean checkout and review the numerical-equivalence table before commit.
